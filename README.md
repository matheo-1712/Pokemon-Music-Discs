# Pokemon-Music-Discs

## Important

Les crafts ne se chargent que si ce pack est installé **comme datapack dans le monde** (`saves/<monde>/datapacks`) puis rechargé avec `/reload`.
Le dossier resource pack sert pour les textures et les modèles.

## Crafts des disques

Chaque disque se craft avec **8 papiers + 1 ingrédient central** et donne un `music_disc_13` avec sa musique, son nom et sa texture propres.
